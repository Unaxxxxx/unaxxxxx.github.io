## 🚀 Getting Started
Welcome to the getecz-laravel-installer! This is a simple installer wizard designed to help you set up Laravel applications without the need for programming knowledge. Follow these steps to get started.

## 📋 System Requirements
Before downloading, ensure your system meets the following requirements:

- **Operating System:** Windows 10 or later, macOS Mojave or later
- **PHP Version:** PHP 8.0 or higher
- **Web Server:** Apache or Nginx
- **Database:** MySQL, MariaDB, or SQLite

## ❓ Troubleshooting
If you encounter issues during installation, consider these common solutions:

- **Installation Fails:** Ensure your system meets the requirements. Verify your PHP version.
